# Code Review Checklist

- Requirements are correctly implemented (MUST)
  - All user stories addressed
  - Acceptance criteria met
  - Edge cases handled
  <!-- Verify that all requirements have been properly implemented -->

- Code follows established patterns and conventions (GOOD)
  - Architecture patterns followed
  - Design patterns applied
  - Code style consistent
  <!-- Code should follow established architectural and design patterns -->

- Performance considerations are addressed (GOOD)
  - Database queries optimized
  - Memory usage reasonable
  - Response times acceptable
  <!-- Code should perform well under expected load conditions -->
