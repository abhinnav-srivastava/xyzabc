# Correctness

  - **How to Measure:** Build the project and check for compilation errors

- **RECOMMENDED:** Verify that common user paths don't throw unexpected runtime exceptions.
  - **How to Measure:** Test main user flows and edge cases

- **RECOMMENDED:** Ensure the code properly handles edge cases and invalid inputs gracefully.
  - **How to Measure:** Test with invalid inputs and boundary conditions
