# Code Review Checklist

- Architecture aligns with system design (MUST)
  - Follows established architecture patterns
  - Integrates properly with existing systems
  - Maintains separation of concerns
  <!-- Implementation should align with the overall system architecture -->

- Scalability and maintainability considered (GOOD)
  - Performance bottlenecks identified
  - Future extensibility planned
  - Code maintainability ensured
  <!-- Design should support future growth and maintenance -->

- Security best practices implemented (MUST)
  - Input validation implemented
  - Authentication/authorization proper
  - Sensitive data protected
  <!-- Security considerations should be properly addressed -->
